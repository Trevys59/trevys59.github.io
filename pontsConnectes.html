Description du projet

Projet de 6 mois sur la réalisation d'une application mobile de réservation liée à la base de données avec une API RESTful. Mise en place de capteurs de surveillance d'ouvrages communicant avec la base de données via LoRaWAN. Gestion de projet et conception de la base de données ont été des aspects clés de ce projet.

Technologies utilisées

  • API RESTful
  • LoRaWAN
  • Java
  • SQL
